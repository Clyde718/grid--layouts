Максимальная ширина контейнера 1280px



1. Делаем основной wrapper который будет грид контейнером с четырьмя  строками(которые имеют высоту авто)

2. Первая строка также будет грид контейнером (в 5 колонок) (авто, фикс-ширина, 1fr, 1fr, 1fr).
   Высота первой строки НЕ задаётся, так высоту определяет первая колонка с контентом (меню) 

3. Вторая строка делится на две фракции (также грид контейнер). Во второй строке высота не задаётся, так как её формирует картинка во второй колонке.
   Первая колонка второй строки также будет грид контейнером с тремя колонками и двумя строками.

4. Есть также третья строка с заголовком (там высота определяется падингами и контентом). 

5. Последняя строка будет грид контейнером с тремя колонками и двумя троками (из этих трёх колонок - первая имеет фиксированную ширину,
   а две остальные по 1fr). Высота строк также задаётся явно (или px или vh) 

6. Футер это также грид контейнер с двумя колонками и одной строкой, внутри каждой из колонок текст размещается с помощью text-align.
